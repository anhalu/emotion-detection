# emotion-detection

clone github 
```
git clone https://github.com/anhalu/emotion-detection.git
cd emotion-detection
```

setup environment 
```
conda create -n emotion_detection python=3.9 
conda activate emotion_detection
pip install -r requirement.txt
```
